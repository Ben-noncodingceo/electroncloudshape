# 🚀 快速入门指南

## 立即使用（3 步）

### 1️⃣ 打开文件
双击 `index.html` 文件，用浏览器打开即可！

### 2️⃣ 选择轨道
- 在左侧选择量子数：n（主量子数）、l（轨道类型）、m（磁量子数）
- 电子云会自动更新

### 3️⃣ 探索
- 用鼠标拖动 3D 视图旋转电子云
- 调整粒子数量和颜色
- 查看右侧的波函数公式

## 🎯 推荐探索路径

### 初学者
1. **1s 轨道**：最简单的球形轨道
   - n=1, l=0(s), m=0
2. **2p 轨道**：经典哑铃形
   - n=2, l=1(p), m=0
3. **3d 轨道**：复杂的四叶草形
   - n=3, l=2(d), m=0

### 进阶探索
- 对比不同 m 值：观察轨道取向变化
- 对比不同原子：H vs He⁺ vs Li²⁺
- 观察节点：2s、3s 的径向节点

## ⚙️ 推荐设置

| 项目 | 值 | 说明 |
|------|-----|------|
| 粒子数量 | 30,000 | 平衡性能和质量 |
| 透明度 | 0.6 | 清晰可见 |
| 粒子大小 | 2.0 | 适中 |

## 🎮 控制说明

**3D 视图**
- 🖱️ 左键拖动 = 旋转
- 🖱️ 右键拖动 = 平移
- 🖱️ 滚轮 = 缩放

**2D 视图**
- 自动显示 XY 平面截面
- 颜色越亮 = 电子概率越大

## ❓ 常见问题

**Q: 为什么看不到电子云？**
A: 检查粒子数量是否太少，或透明度是否太低

**Q: 如何让电子云更清晰？**
A: 增加粒子数量到 50,000 以上

**Q: 性能太慢怎么办？**
A: 减少粒子数量到 10,000-15,000

**Q: 2D 图看起来很粗糙？**
A: 这是正常的，2D 是截面投影，不是完整的电子云
